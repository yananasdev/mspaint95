# mspaint95
Canvas drawing in javascript, multiple functionalities. VanillaJS.
Check it out on [Codepen](https://codepen.io/yananas/pen/rwvZvY).


## Functionalities
- Color change
- Eraser
- Clear canvas
- Change canvas size
- Brush size change
- Save/load function
- Secret button
